# flickr_gallery_app
It's an Android photo feed app using the Flickr API to test some technologies like:

* MVVM 
* Koin
* Kotlin Coroutines
* Pagging
* Room
* Android Jetpack Architectural Components

# Instructions
To run the application, you will need to obtain an API key from Flickr [here](https://www.flickr.com/services/api/misc.api_keys.html) and create a file at the root of the project called ```keys.properties``` containing this
```js
 API_KEY="your_flickr_api_key"
```
